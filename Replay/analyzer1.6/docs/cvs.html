How to access the C++ analyzer CVS repository

Anyone with a JLab account has read-only access to the CVS repository. Access is available both for on-site and off-site users. To check out the current version, issue the following commands: 
export CVS_RSH=ssh
cvs -d :ext:cvs.jlab.org:/group/halla/analysis/cvs checkout analyzer
This will create a directory called "analyzer" containing your working copy of the CVS tree.

For more information on CVS, see the CVS home page. Excellent documentation can be found in the CVS Book.

Contact Ole Hansen if you would like to be added to the list of people with write access to the repository.
